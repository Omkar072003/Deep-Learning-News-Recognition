# Deep Learning Content Recognition - Frontend

This is the frontend app for the media content recognition project. It is built as a lightweight, modular static site and communicates with the Node.js backend AI API.

## Key Features
- Drag-and-drop image/video upload
- Real-time results visualization
- Responsive design for all devices
- RESTful API integration for media analysis

## Getting Started
- Install dependencies: `npm install`
- Start dev server: `npm run start`
