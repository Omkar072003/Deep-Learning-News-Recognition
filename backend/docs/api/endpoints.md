# API Endpoints

## /api/v1/analyze/image
POST - Analyze the content of an image file.

## /api/v1/analyze/video
POST - Analyze video content.

## /api/v1/upload
POST - Upload a file (image/video).

## /api/v1/results
GET - Get previous analysis results.

## /api/v1/health
GET - Check server health status.
