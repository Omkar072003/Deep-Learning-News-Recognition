# Deep Learning Content Recognition

This repository contains a fullstack implementation of a deep learning based system for automated news content analysis (images and videos).

## Features

- Upload and analyze images/videos using advanced CNN and RNN architectures
- REST API backend using Node.js, Express, TensorFlow.js, and SQLite
- Frontend web app for easy upload and visualization
- Docker-based deployment and development workflow
- Modular codebase for easy extension and research

## Setup

1. Install dependencies in `backend` and `frontend`
2. Run database setup script (`node scripts/setup/database-setup.js`)
3. Fill out `.env` files as needed
4. Start backend and frontend (`npm start`)
5. For Docker, use `docker-compose up --build`
